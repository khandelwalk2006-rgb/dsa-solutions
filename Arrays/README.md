# Arrays - DSA Solutions

This folder contains basic and important array problems solved in C++.
These problems are useful for interview preparation and strengthening
fundamental concepts of Data Structures.

## Topics Covered
- Reverse an Array (Two Pointer Approach)
- Find Minimum and Maximum in an Array
- Check if Array is Sorted
- Move Zeros to End
- second largest number
