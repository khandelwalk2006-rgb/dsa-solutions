# Recursion

This folder contains recursion-based problems and their implementations in C++.

## Topics Covered

- Binary Search using Recursion
- Bubble Sort using Recursion
- Selection Sort using Recursion
- Insertion Sort using Recursion
- Book Allocation using Recursion
- Search in Rotated Sorted Array
- Find Minimum in Rotated Sorted Array
- Square Root of a Number
- Reverse String using Recursion
- Check Palindrome using Recursion
- Fibonacci Number
- Sum of Elements using Recursion
- Peak Index in a Mountain Array
- First and Last Position of Element

## Language Used
- C++

## Purpose
To improve understanding of recursion, recursive thinking, and problem-solving in Data Structures and Algorithms.
